## Setup

```shell
yarn install
```

# Usage

```shell
yarn start
yarn build
yarn test
yarn test:ci
yarn test:cover
yarn lint
yarn prettier
yarn prettier:watch
yarn checktypes
yarn checktypes:build
```

# VSCode

Recommended plugins:

- ESLint
- Bracket Pair Colorizer

Add the following config on vscode settings.json to enable typescript eslint IDE integration:

```json
"eslint.validate": [
  "javascript",
  "javascriptreact",
  "typescript",
  "typescriptreact"
]
```
